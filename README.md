# World_Game
This repo contains a basic terminal-based version of the newly popular **WORDLE**. The implementation is tried to be as professional and hard coded to evaluate basics knowledge during the journey of learning C++ and OOPs. 
## 🚀 Features
- **Custom Word List:** Loads words from an external `Data.csv` file.
- **Color-Coded Feedback:** Uses terminal colors to show correct letters, misplaced letters, and wrong guesses.
- **Efficient Logic:** Utilizes a header file for clean macro management.

## 🛠️ Requirements
- A C++ compiler (like `g++` or Clang).
- A terminal that supports ANSI color codes (most modern terminals).
